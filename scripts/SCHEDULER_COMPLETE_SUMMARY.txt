WHAT THE SCHEDULER DOES
================================================================================

Every 5 minutes automatically:
   1. Check AI_Employee/vault/Inbox/ for new tasks
   2. Run task-planner.py to analyze tasks
   3. Generate execution plans
   4. Save plans to AI_Employee/vault/Needs_Action/
   5. Log all operations to logs/scheduler.log
   6. Update statistics in logs/scheduler_registry.json

This means your AI Employee will continuously:
   ✓ Monitor for new work
   ✓ Plan tasks automatically
   ✓ Prepare work for execution
   ✓ Never miss incoming tasks

================================================================================
TROUBLESHOOTING
================================================================================

SCHEDULER NOT RUNNING:

Windows:
   1. Open Task Scheduler (Win+R > taskschd.msc)
   2. Look for "AI Employee Scheduler"
   3. Check if enabled and last run status
   4. Run setup_scheduler.bat again as Administrator

Linux/Mac:
   1. Check cron job: crontab -l
   2. Verify cron is running: sudo systemctl status cron
   3. Check logs: tail -f logs/scheduler.log
   4. Run setup: bash setup_scheduler.sh

TASKS NOT PROCESSING:

   1. Verify vault exists: ls AI_Employee/vault/Inbox/
   2. Check logs: tail -f logs/scheduler.log
   3. Run health check: python run_ai_employee.py --health
   4. Test manually: python run_ai_employee.py --once --verbose
   5. Check permissions: ls -la AI_Employee/scripts/

PERMISSION DENIED:

Windows:
   - Run setup_scheduler.bat as Administrator
   - Check user has write access to directories

Linux/Mac:
   - Make executable: chmod +x run_ai_employee.py
   - Check directory permissions: ls -la
