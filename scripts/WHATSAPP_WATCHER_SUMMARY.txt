================================================================================
WHAT HAPPENS
================================================================================

1. Browser Launch:
   - Chromium opens in headless mode
   - Navigates to web.whatsapp.com
   - Session loads (or QR code appears first time)

2. Monitoring:
   - Checks every 60 seconds
   - Extracts unread chats
   - Checks message text for keywords
   - Creates action files for matches

3. Action Files Created:
   - Location: vault/Needs_Action/
   - Format: WHATSAPP_[chat_name]_[timestamp].md
   - Content: YAML frontmatter + markdown with message and actions

4. Continuous Operation:
   - Runs forever checking every minute
   - Press Ctrl+C to stop gracefully
   - All logs preserved in vault/whatsapp_watcher_log.txt

================================================================================
CONFIGURATION OPTIONS
================================================================================

To modify the watcher, edit whatsapp_watcher.py:

1. Keywords (default: urgent, asap, invoice, payment, help, sales, quote)
   Find: KEYWORDS = [...]
   Change to custom list

2. Check Interval (default: 60 seconds)
   Find: time.sleep(60)
   Change to desired seconds (e.g., 30, 120)

3. Headless Mode (default: True = hidden browser)
   Find: headless=True
   Change to headless=False to see browser (debug)

4. Log Level (default: INFO)
   Find: logger.setLevel(logging.INFO)
   Change to DEBUG, WARNING, or ERROR

================================================================================
TROUBLESHOOTING QUICK FIXES
================================================================================

"ModuleNotFoundError: No module named 'playwright'"
  → pip install -r requirements_whatsapp.txt

"No module named 'playwright._impl'"
  → playwright install chromium

"Browser doesn't launch"
  → Check: playwright install chromium
  → Check: Admin privileges
  → Check: Disk space

"QR code timeout"
  → Extend timeout in code
  → Check internet
  → Try again later

"No action files created"
  → Check: Unread messages in WhatsApp
  → Check: Messages contain keywords
  → Review: vault/whatsapp_watcher_log.txt
  → Send test message with keyword

"Selector not found"
  → WhatsApp UI may have changed
  → Run with headless=False to debug
  → Update selectors in code
