┌────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                            │
└────────────────────────────────────────────────────────────────────────────┘

Problem: "Playwright not installed"
→ pip install playwright && playwright install chromium

Problem: "Browser window didn't open"
→ Check: playwright install chromium
→ Check: System has enough RAM/disk space

Problem: "Timeout loading LinkedIn"
→ Check internet connection
→ Try again (sometimes temporary)
→ LinkedIn may have blocked automation

Problem: "Login required" on 2nd run
→ Delete session: rm -r vault/linkedin_session/
→ Login again: python scripts/linkedin_watcher.py

Problem: "No messages found"
→ This is NORMAL - no new unread messages detected
→ Script waits 5 minutes and checks again

Problem: "File creation failed"
→ Check vault/Needs_Action/ permissions
→ Check disk space available
→ Check logs for details
