VERIFY SUCCESS
================================================================================

Check Recipients' Inbox:
   1. Look in recipient's inbox (not spam!)
   2. Subject should match what you sent
   3. Body should contain your message
   4. Sender should be your email address

   1. Wait a few minutes (mail delay is normal)
   2. Check spam folder
   3. Verify recipient email is correct
   4. Check skill output for errors

================================================================================
TROUBLESHOOTING
================================================================================

ERROR: "EMAIL_ERROR: Authentication failed"
   Fix:
   - Check EMAIL_ADDRESS is correct
   - Verify using App Password (not regular password)
   - Enable 2-Step Verification on Gmail account
   - Verify credentials: echo %EMAIL_ADDRESS%

ERROR: "EMAIL_ERROR: Connection refused"
   Fix:
   - Check internet connection
   - Check firewall allows port 587
   - Try different SMTP settings
   - Verify SMTP server is accessible

ERROR: "Parameter required: --to, --subject, --body"
   Fix: Include all three parameters:
   - --to recipient@example.com (REQUIRED)
   - --subject "Your Subject" (REQUIRED)
   - --body "Email text" (REQUIRED)
   - --cc optional@example.com (OPTIONAL)

ERROR: Email not appearing in inbox
   Fix:
   - Check spam/junk folder
   - Wait a few minutes
   - Verify recipient email address
   - Check sender not on blocklist

================================================================================
ADVANCED: BATCH EMAIL SCRIPT
================================================================================

Create file: send_batch_emails.bat

Content:
   @echo off
   setlocal enabledelayedexpansion

   REM Define recipients
   set RECIPIENTS=user1@example.com user2@example.com user3@example.com
   set SUBJECT=Important Announcement
   set BODY=Dear Customer, Important update text here...

   REM Send to each recipient
   for %%R in (%RECIPIENTS%) do (
       echo Sending to %%R...
       python .claude/skills/gmail-send/scripts/send_email.py ^
         --to %%R ^
         --subject "%SUBJECT%" ^
         --body "%BODY%"
   )

   echo All emails sent!

Run it:
   send_batch_emails.bat
