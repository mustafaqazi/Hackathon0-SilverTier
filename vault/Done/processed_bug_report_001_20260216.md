## Bug Report Details

**Title:** Login button not working on mobile devices

**Severity:** HIGH - Blocks user access

**Description:**
The login button is unresponsive on mobile Safari and Chrome browsers. Users cannot authenticate on mobile devices.

**Steps to Reproduce:**
1. Open app on iPhone/Android
2. Navigate to login page
3. Click login button
4. Button does not respond

**Expected Behavior:**
Login modal should open

**Actual Behavior:**
Button click has no effect

**Environment:**
- iOS 15.4+
- Android 11+
- Mobile Safari and Chrome
